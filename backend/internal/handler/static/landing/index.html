How it works

Three components, one seamless experience.

1

Install the Mac agent

A lightweight menu-bar app that watches your Claude Code sessions and relays events securely.

2

Sign in on your iPhone

Download AFK from the App Store, sign in, and pair with your Mac. Keys are exchanged automatically.

3

Walk away

Leave your desk. Your sessions stream live to your pocket. Approve, prompt, or just watch.

Security first

Your code is sensitive. AFK is built with that in mind.

🔐

E2E Encryption

X25519 ECDH key exchange, AES-256-GCM content encryption. The server never sees plaintext.

✍️

Ed25519 Signing

Every command is cryptographically signed. Replay attacks are impossible.

🚫

Zero Knowledge

The relay server routes encrypted blobs. No logs, no storage, no access to your data.

🍎

Apple Keychain

All keys stored in the Secure Enclave via Keychain. Never written to disk.
